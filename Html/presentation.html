Introduction

Ce site est un jeu de piste nommé thunder avec des super-héros qui cherchent l'amour.

Le but:


Le but du jeu est de trouver les couples de super-héro qui vont ensemble Votre but à vous, joueur, est de trouver le code final dans la page login pour gagner.

Le déroulement:


Les super-héros sont numérotés de 1 à 10. L'ordre des personnages dans l'ordre du site. DR. DAVID, TROLATOR, CHICKENMAN, TELEPORTGIRL, PUGMAN, IGOR, I-SCREAM, COOLGIRL, CAPTAINBALDASS, ELEPHANTMAN.

Comment obtenir le code ?


Assembler les deux nombres de chaque couple par ordre croissant pour obtenir le code du couple. Ranger tous les codes de couple par ordre croissant et valider la réponse. (ex : 12 – 34 -56 -78- 910)
Maintenant à vous de jouer joueur!
